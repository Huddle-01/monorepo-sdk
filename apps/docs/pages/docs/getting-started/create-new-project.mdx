# Create a New Project using Huddle01 CLI

> If you have not already installed the Huddle01 CLI, please follow the instructions in the [Getting Started](/docs/getting-started) section.

If you want add Huddle01 to your project, you can use the Huddle01 CLI to create a new project with the things you need to get started.
this will create a new project all the configuration files and the necessary dependencies to use Huddle01 infrastructure.
Replace the code to tailor the application to meet your requirements.

> **NOTE:** The Project will be created in the current directory, and will be using `React` as the frontend framework, and `VITE` as the build tool.

## Create a New project

To get started with the CLI Tool, you need to open a terminal and run the following command:

```bash
npx create-huddle01
```

This will create a new project with the following structure:

```bash
.
├── .env
├── .env.example
├── .gitignore
├── README.md
├── package.json
├── public
│   ├── favicon.ico
│   └── index.html
├── src
│   ├── App.vue
│   ├── assets
│   │   └── logo.png
│   ├── components
│   │   └── HelloWorld.vue
│   ├── main.js
│   └── router.js
└── vite.config.js
```
