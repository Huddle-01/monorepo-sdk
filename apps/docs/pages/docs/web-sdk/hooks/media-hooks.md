# Client Media Hooks

## `useMeCamTrack`

### Description

Hook to access the camera track of the current user
<br/>
<font size="4"> **Return Type : `Promise<MediaTrack>`** </font>

```jsx
// JavaScript | TypeScript
import { useMeCamTrack } from "huddle01-client/hooks";

// Example
const myCam = useMeCamTrack();

```
