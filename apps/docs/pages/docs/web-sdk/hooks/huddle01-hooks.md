# Huddle01 Hooks

## `useHuddleStore`

### Description

Hook to access HuddleStore states, mutation and accessor methods

```jsx
// JavaScript | TypeScript
import { useHuddleStore } from "huddle01-client/hooks";

// Example
const isCamPaused = useHuddleStore((state) => state.isCamPaused);
```
